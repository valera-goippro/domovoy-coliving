# domovoy-coliving

Production landing page for [coliving.smartcare.house](https://coliving.smartcare.house) — a coliving home near Porto, Portugal.

A single self-contained HTML file (all CSS, JS and SVG inline, no external dependencies).

## About

Domovoy is a smart home near Porto open to remote workers, makers and families.
No rent — in exchange for everyday company and a hand with small things.

## Deployment

The file `index.html` is the docroot. Drop it behind any static web server or CDN.
Current staging docroot: `/home/administrator/coliving-web/`
